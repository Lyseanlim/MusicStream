# Project Description

MusicStream is a music streaming website that we developed using Django. Users can create an account, log in, browse songs, search for music, create playlists, save favorite songs, and listen to music online.

This project was created for our Django Web Development course. It helped us understand how to build a complete web application using Django, connect a database, manage user accounts, and organize a project into different apps.

# How to Run the Project

1. Install the required packages.

```
pip install -r requirements.txt
```

2. Run database migrations.

```
python manage.py migrate
```

3. Start the server.

```
python manage.py runserver
```

4. Open your browser and visit:

```
http://127.0.0.1:8000/
```
